# FHIR-Converter
Freiwilliges Projekt im Fach Grundlagen von eHealth an der Hochschule Kempten

## Table of contents

- [Features](#features)

## Features
- convert HL7v2 Messages to HL7 FHIR standard
- fetch and interact with patient data from a server (GET, POST, PUT)
